## Getting Started

1. **Create a GitHub account** (if you don't have one)
   - Go to [GitHub](https://github.com/signup) to sign up

2. **Navigate to Discussions**
   - Visit the [MCP Studio Discussions](https://github.com/sandraschi/mcp-studio/discussions)
   - Click "New discussion" to start a new conversation

3. **Choose a category**
   - Select the most appropriate category for your post
   - See below for category descriptions

## Posting Guidelines

### Do:
- Be respectful and inclusive
- Use clear, descriptive titles
- Format code using markdown code blocks
- Search before posting
- Mark the best answer if your question is resolved

### Don't:
- Post duplicate questions
- Share sensitive information
- Spam or self-promote excessively
- Be rude or disrespectful

## FAQ

### How do I format code in my posts?
Use triple backticks (```) before and after your code block:

````markdown
```python
def hello():
    print("Hello, MCP Studio!")
```
````

### How do I report a bug?
Please use the [issue tracker](https://github.com/sandraschi/mcp-studio/issues) for bug reports.

### Can I help moderate discussions?
Yes! Active community members may be invited to become moderators. Start by being helpful and following the guidelines.

## Moderation

Our goal is to maintain a welcoming and productive environment. Moderators may:
- Move discussions to more appropriate categories
- Edit or remove content that violates guidelines
- Temporarily or permanently ban users who repeatedly violate rules

## Code of Conduct

All participants must follow our [Code of Conduct](CODE_OF_CONDUCT.md). Please report any violations to the maintainers.
